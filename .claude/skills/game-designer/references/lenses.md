# Lenses Catalog

Apply 2–4 lenses at a time. Each is a question to put to the design.

## Experience Lenses

| Lens                 | Core question                                                          |
| -------------------- | ---------------------------------------------------------------------- |
| Essential Experience | What experience do I want? What is essential to it?                    |
| Surprise             | What surprises players? What's predictably dull that shouldn't be?     |
| Fun                  | What specifically is fun? What isn't? Why?                             |
| Curiosity            | What questions does the game put in the player's mind?                 |
| Endogenous Value     | What is valuable _inside_ the game world? Is it perceived as valuable? |
| Pleasure             | What pleasures does this game provide? What pleasures am I missing?    |

## Structure Lenses

| Lens                | Core question                                                        |
| ------------------- | -------------------------------------------------------------------- |
| Elemental Tetrad    | Are mechanics, story, aesthetics, and technology serving each other? |
| Unifying Theme      | Is every element expressing the same core idea?                      |
| Problem Statement   | What problem am I actually solving? What are my assumptions?         |
| Elegance            | What is the simplest version that preserves the essence?             |
| Inner Contradiction | Are any elements working against each other?                         |
| Character           | Does the aesthetic feel unified and uniquely itself?                 |

## Mechanics Lenses

| Lens               | Core question                                                                    |
| ------------------ | -------------------------------------------------------------------------------- |
| Rules              | Are the rules clear? Do they create the intended experience?                     |
| Skill vs. Chance   | Is the balance of skill and luck right for this audience?                        |
| Fairness           | Does the game feel fair? Is it actually fair? Are these different?               |
| Meaningful Choices | Do decisions have real trade-offs with no dominant option?                       |
| Triangularity      | Are there low-risk/low-reward AND high-risk/high-reward options?                 |
| Economy            | Are resources flowing correctly? Is there scarcity and abundance where intended? |
| Time               | Does the time structure create the right tension and pacing?                     |
| Visible Progress   | Can players see their progress? Does it feel rewarding?                          |

## Player Lenses

| Lens          | Core question                                                               |
| ------------- | --------------------------------------------------------------------------- |
| The Player    | Who is my player? What do they want? What skills do they value?             |
| Motivation    | What motivates the player to keep playing right now?                        |
| Reward        | What rewards does the game give? Are they the right type at the right time? |
| Flow          | Is the challenge/skill curve keeping players in the flow channel?           |
| Feedback      | Does the game respond clearly and satisfyingly to every player action?      |
| Juiciness     | Does every interaction feel alive and gratifying?                           |
| Accessibility | Who is being excluded by the current design? Is that intentional?           |

## Story & World Lenses

| Lens                     | Core question                                                            |
| ------------------------ | ------------------------------------------------------------------------ |
| Story Machine            | Is the game generating emergent stories worth sharing?                   |
| Hero's Journey           | Does the progression arc mirror the monomyth?                            |
| Character Transformation | Does the player (or player character) change meaningfully?               |
| World                    | Does the world feel real, consistent, and worth exploring?               |
| The Obstacle             | Are the obstacles between the player and their goal the right obstacles? |

## Process Lenses

| Lens              | Core question                                                         |
| ----------------- | --------------------------------------------------------------------- |
| The Loop          | How fast can I iterate? What's slowing the loop?                      |
| Prototype         | What is the cheapest version of this I could test today?              |
| The Playtester    | What are playtesters actually experiencing vs. what I think they are? |
| Problem Statement | Have I clearly defined the problem, or am I solving the wrong thing?  |

## Ethics & Purpose Lenses

| Lens           | Core question                                                       |
| -------------- | ------------------------------------------------------------------- |
| Responsibility | Am I comfortable with the effect this game will have on players?    |
| The Client     | Am I serving real needs, not just stated needs?                     |
| The Designer   | Why am I making this? Am I making it with genuine care for players? |
| Transformation | Will players be more alive, aware, or connected after playing?      |

---

## Lens Selection Guide

| Design symptom                         | Lenses to apply                                                 |
| -------------------------------------- | --------------------------------------------------------------- |
| Game feels hollow or purposeless       | Essential Experience, Unifying Theme, The Designer              |
| Players stop playing too soon          | Flow, Motivation, Reward, Visible Progress                      |
| Something feels unfair                 | Fairness, Skill vs. Chance, Feedback                            |
| Mechanics feel arbitrary               | Rules, Meaningful Choices, Elegance                             |
| Story and gameplay fighting each other | Story Machine, Inner Contradiction, Elemental Tetrad            |
| Game is fun but not memorable          | Surprise, Character Transformation, World                       |
| Unsure if design is ready              | The Eight Filters (see SKILL.md), Playtester, Problem Statement |
