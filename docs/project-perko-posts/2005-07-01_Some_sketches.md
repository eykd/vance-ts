## Friday, July 01, 2005 


### Some sketches

I'm trying to get the hang of using the tablet for pencilling and inking, as well as trying to find some fast and decent-looking way of coloring. I did these last night.  
  
[Here's](http://www.projectperko.com/images/light.jpg) a screwed-up attempt to find some decent coloring method. I could shave this type of coloring down to probably fifteen-twenty minutes per element of complexity, but it doesn't look nearly as good as the old versions because I'm not picking and choosing the different shades of color for each color swatch. If you have opencanvas, you can see the core files [here](http://www.projectperko.com/images/light.oci) and [here](http://www.projectperko.com/images/light.oe3).  
  
[Here's](http://www.projectperko.com/images/bishie.jpg) a bishie I sketched for kicks, and [here's](http://www.projectperko.com/images/goth.jpg) a goth I sketched just for kicks. Careful: these are FULL SIZE. The goth, in particular, is half a meg. That's what hatched shading gets you.
