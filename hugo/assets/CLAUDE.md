# Assets

Source files processed by Hugo Pipes. Supports CSS/JS bundling, minification, and fingerprinting.

## Subdirectories

- `css/` - TailwindCSS source with DaisyUI imports

## Skills

- `/tailwind-daisyui-design` - Component styling
- `/daisyui-design-system-generator` - Theme creation

## Notes

- `styles.css` is the TailwindCSS entry point
- Hugo Pipes processes with PostCSS at build time
- Output is fingerprinted for cache busting
