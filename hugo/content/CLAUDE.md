# Content

Markdown content files organized by section. Hugo generates pages from these files using layouts.

## Structure

- `_index.md` - Home page content
- `posts/` - Blog posts section

## Conventions

- Front matter in YAML format
- Section index files named `_index.md`
- Individual pages use any `.md` filename
- Taxonomies: tags, series

## Skills

- `/hugo-templates` - Content rendering
- `/hugo-search-indexing` - Making content searchable
